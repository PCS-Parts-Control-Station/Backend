취소

입고 등록 흐름

현재 입력 내용은 입고 전표와 품목별 재고 변화 기록으로 저장됩니다.

  1. 입고 등록

    거래처와 입고 품목을 입력하고 관리번호 발급 대상을 확인합니다.

    1. 1
      기본 정보 입력

      거래처와 입고 사유를 입력합니다.

    2. 2
      품목 검색

      품목을 찾아 수량과 사유를 입력합니다.

    3. 3
      품목 검토

      추가된 품목을 확인하고 수정합니다.

    4. 4
      검수 전

      저장 후 관리번호가 발급됩니다.

  2. 검수

    등급과 판매 가능 여부를 확정합니다.

  3. 출고

    판매 가능 품목을 출고합니다.
