취소

입고 등록 흐름

현재 입력 내용은 입고 전표와 부품별 재고 변화 라인으로 저장됩니다.

  1. 입고 등록

    거래처와 부품 라인을 입력하고 관리번호 발급 대상을 확인합니다.

    1. 1
      거래처 선택

      공급 거래처를 전표에 연결합니다.

    2. 2
      부품 라인 입력

      부품과 입고 수량을 라인별로 입력합니다.

    3. 3
      관리번호 발급

      수량만큼 개별 부품 관리번호가 생성됩니다.

    4. 4
      검수 대기

      입고 부품은 검수 대기로 넘어갑니다.

  2. 검수

    등급과 판매 가능 여부를 확정합니다.

  3. 출고

    판매 가능 부품을 출고합니다.

등록 규칙

  • 전표 저장 후 원본 수량은 직접 수정하지 않음
  • 입고 취소는 취소 전표와 취소 라인으로 보존
  • 제조사 시리얼은 추후 개별 부품 상세에서 보강 가능
