입고 등록

거래처, 입고 사유, 부품 라인을 입력해 입고 전표를 생성합니다.

전표 목록

1 전표 기본 정보

전표번호는 저장 시 자동 발급됩니다.

작성 중

2 부품 검색

부품관리의 마스터 정보를 검색한 뒤 수량과 라인 사유를 입력해 전표에 추가합니다.

부품명 모델 / 코드 카테고리 선택

선택한 부품 부품을 검색해 주세요

검색 결과에서 부품을 선택하면 여기에 표시됩니다.

3 부품 라인

전표에 추가된 부품 라인입니다. 수량과 라인 사유를 수정하거나 삭제할 수 있습니다.

0개 라인

검색한 부품을 추가하면 여기에 표시됩니다.

4 저장 후 처리

입고 저장 시 전표와 재고 변화 라인이 생성되고, 개별 부품은 검수 대기와 판매 보류 상태로 시작합니다.

취소

입고 등록 흐름

현재 입력 내용은 입고 전표와 부품별 재고 변화 라인으로 저장됩니다.

  1. 입고 등록

    거래처와 부품 라인을 입력하고 관리번호 발급 대상을 확인합니다.

    1. 1
      기본 정보 입력

      거래처와 입고 사유를 입력합니다.

    2. 2
      부품 검색

      부품을 찾아 수량과 사유를 입력합니다.

    3. 3
      라인 검토

      추가된 라인을 확인하고 수정합니다.

    4. 4
      검수 대기

      저장 후 관리번호가 발급됩니다.

  2. 검수

    등급과 판매 가능 여부를 확정합니다.

  3. 출고

    판매 가능 부품을 출고합니다.

등록 규칙

  • 전표 저장 후 원본 수량은 직접 수정하지 않음
  • 입고 취소는 취소 전표와 취소 라인으로 보존
  • 제조사 시리얼은 추후 개별 부품 상세에서 보강 가능
