입고부터 출고까지

현재 화면은 재고 흐름의 입고 단계입니다.

  1. 입고

    거래처 전표로 품목 입고를 기록합니다.

    1. 1
      거래처 연결

      공급 거래처를 선택합니다.

    2. 2
      입고 품목

      품목과 수량을 추가합니다.

    3. 3
      관리번호 발급

      수량만큼 개별 번호를 만듭니다.

    4. 4
      검수 전

      검수 전 상태로 보관합니다.

  2. 검수

    등급과 판매 가능 여부를 정합니다.

  3. 출고

    판매 가능 품목을 전표로 출고합니다.

  4. 이력

    전표와 상태 변경 기록을 확인합니다.

입고 처리 기준

  • 입고 완료 시 수량만큼 관리번호를 생성
  • 개별 부품은 검수 전, 판매 보류로 시작
  • 입고 오류는 원본 수정 대신 취소 전표로 보존
