입고부터 출고까지

현재 화면은 재고 흐름의 검수 단계입니다.

  1. 입고

    거래처 전표로 부품 입고를 기록합니다.

  2. 검수

    전표에서 관리번호를 선택하고 결과를 등록합니다.

    1. 1
      전표 선택

      검수할 입고 전표를 고릅니다.

    2. 2
      대상 선택

      관리번호를 개별 또는 일괄 선택합니다.

    3. 3
      결과 등록

      등급과 판매상태를 저장합니다.

    4. 4
      이력 확인

      최근 검수와 정정 기록을 확인합니다.

  3. 출고

    판매 가능 부품을 전표로 출고합니다.

  4. 이력

    전표와 상태 변경 기록을 확인합니다.

검수 처리 기준

  • 검수 결과는 원본 수정 대신 새 이력으로 저장
  • 불량 등급은 판매 불가 상태로 반영
  • 상태 변경 시 부품 상태 이력을 함께 저장
