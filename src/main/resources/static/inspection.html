입고부터 출고까지

현재 화면은 재고 흐름의 검수 단계입니다.

  1. 입고

    거래처 전표로 부품 입고를 기록합니다.

  2. 검수

    등급과 판매 가능 여부를 정합니다.

    1. 1
      대상 선택

      검수가 필요한 개별 부품을 선택합니다.

    2. 2
      결과 입력

      항목별 상태와 메모를 기록합니다.

    3. 3
      등급 확정

      판매 판단에 사용할 등급을 정합니다.

    4. 4
      상태 반영

      판매 가능 여부와 상태 이력을 저장합니다.

  3. 출고

    판매 가능 부품을 전표로 출고합니다.

  4. 이력

    전표와 상태 변경 기록을 확인합니다.

검수 처리 기준

  • 검수 결과는 원본 수정 대신 새 이력으로 저장
  • 불량 등급은 판매 불가 상태로 반영
  • 상태 변경 시 부품 상태 이력을 함께 저장
