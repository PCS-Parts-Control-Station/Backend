# PCS Handoff

이 문서는 일시적인 작업 인수인계만 기록한다. 영구 규칙, 전체 파일 목록, 완료된 작업 내역은 두지 않는다.

## 읽기 순서

1. `AGENTS.md`
2. `docs/ai/AI_INDEX.md`
3. 현재 작업을 이어받을 때만 이 문서

## 현재 상태

- 기준일: 2026-07-14
- 영구 구조와 기능 계약은 각 원본 문서를 따른다.
- 최신 자동 검증 결과는 `harness/reports/agent-failures.md`의 생성 시각과 FAIL만 확인한다.
- 이 문서의 내용이 기능·DB·디자인 원본과 충돌하면 원본 문서를 우선하고 이 문서를 수정한다.

## 진행 중 작업

- 없음. 새 작업은 `AI_INDEX.md`에서 문서를 선택해 시작한다.

## 작성 규칙

- 진행 중인 목표, 미완료 항목, 마지막 검증, 즉시 필요한 다음 행동만 적는다.
- 완료된 화면 설명, mock 데이터, 전체 API 목록, DDL 목록, 공통 규칙은 적지 않는다.
- 작업 종료 시 불필요해진 항목을 삭제한다.
